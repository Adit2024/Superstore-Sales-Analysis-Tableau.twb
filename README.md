# Superstore-Sales-Analysis-Tableau.twb
Project Overview

This project focuses on analyzing the Superstore sales dataset using Tableau. The main objective is to transform raw sales data into meaningful insights through interactive dashboards and data storytelling. The project highlights key trends in sales, profit, product categories, and regional performance.

Tools Used

Tableau

Superstore Dataset

Data Visualization Techniques

Dashboard Features

Sales trend analysis over time

Category-wise sales and profit distribution

Regional sales performance using maps

Sub-category comparison through bar charts

Interactive filters for better exploration

Key Insights

Identification of high-performing product categories

Analysis of sales growth across different years

Regional patterns in sales and profit

Business insights to support better decision-making

Conclusion

The Tableau dashboard and story help in understanding business performance through visual analytics. This project demonstrates the ability to create interactive dashboards and communicate insights effectively using data visualization.
